클래스101 코테 문제

- B[k] = 0;을 넣지 않는 실수를 했음
- sum과 배열 B를 통해 가장 작은 음수를 더해가는 식으로 갯수 구하기

```java
static int solution(int[] A) {
		int sum = 0;
		int pt = 0;
		int[] B = new int[A.length];
        for(int i = 0; i< A.length; i++){
        	B[i] = A[i];
          sum += A[i];
          if(sum < 0) {
            Arrays.sort(B);
            int k = 0;
            while(sum<0) {
              sum -= B[k];
              B[k] = 0;
              k++;
              pt++;
            }
          }
        }
        return pt;
    }
```
