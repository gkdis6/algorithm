# 유사도

## 자카드 유사도

- 두 문장을 각각 단어의 집합으로 만든 뒤 두 집합을 통해 유사도를 측정하는 방식 중 하나
- 측정하는 방법은 두 집합의 교집합을 두 집합의 합집합으로 나눠주는 것

abxxo, abxx 일 경우

![img](https://blog.kakaocdn.net/dn/yA9lw/btqBLkikiju/zKIiVLxQV6g4L3eFFB3JDk/img.png)

자카드 유사도는 3/4
